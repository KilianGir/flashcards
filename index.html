📲 Installer l'application sur votre appareil

Android (Chrome)

  1. Ouvrez cette page dans Chrome.
  2. Appuyez sur le menu ⋮ (trois points, en haut à droite).
  3. Sélectionnez « Installer l'application » (ou « Ajouter à l'écran d'accueil »).
  4. Confirmez : une icône apparaît sur votre écran d'accueil.

iPhone / iPad (Safari)

  1. Ouvrez cette page dans Safari (obligatoire — Chrome sur iOS ne permet pas l'installation).
  2. Appuyez sur le bouton Partager (le carré avec une flèche vers le haut).
  3. Faites défiler et sélectionnez « Sur l'écran d'accueil ».
  4. Confirmez : l'app s'ouvrira ensuite en plein écran, sans barre d'adresse.

Ordinateur (Chrome / Edge)

  1. Ouvrez cette page dans Chrome ou Edge.
  2. Cliquez sur l'icône d'installation ⊕ dans la barre d'adresse (à droite).
  3. Cliquez sur « Installer » dans la fenêtre qui s'ouvre.

Une fois installée, l'app fonctionne hors ligne pour tout ce que vous avez déjà consulté en ligne.
